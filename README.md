# Jetris
## ABOUT THE PROJECT
Jetris is a fanmade game of tetris, just a test-made game for the dev to improve on his coding skills!
- Jetris is a prototype python game that has leaderboards and scores just like Tetris

## BUILT WITH:
- Python 3
- Pygame

## GETTING STARTED
Install dependencies using pip

```python
pip install -r requirements.txt
```

Start your Jetris
```python
python main.py
```

## HOW TO PLAY
1. It's Just same as tetris!
2. Just place blocks, and have fun!
